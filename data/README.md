## IMDb Dataset

The data used in this project is sourced from IMDb public datasets.

Due to file size limitations, the raw TSV files are not included in this repository.

### Required Files:
- title.basics.tsv
- title.ratings.tsv
- title.crew.tsv
- title.episode.tsv
- name.basics.tsv

### Download Source:
https://datasets.imdbws.com/

After downloading, place the files inside this `data/` folder.
